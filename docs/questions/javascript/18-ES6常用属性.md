### let & const

相较于之前的 `var` 声明，主要有以下几个区别：

1. 不存在变量提升
2. 存在暂时性死区,预编译时不能被访问
3. 存在块级作用域，,作用域之外不能访问里面声明的变量
4. 不允许重复声明，同一作用域内只能声明一次
5. `const` 声明后不能修改

### Set/Map 对象

```js
let arr = Array.from(new Set([1, 2, 2, 3]));

let set = new Set();
set.add(1);
set.has(1); // true
set.delete(1);
set.clear();

let map = new Map();
map.set('n', 1);
map.get('n');
map.size // 1
map.delete('n');
map.entries();
```
